Dog Breed Detector

  • This application uses convolutional neural network to detect dog breed.
  • If a dog is detected, it will show the breed it most likely is.
  • If a human face is detected, it will show the most similar dog breed.
  • It supports input in png, jpg and jpeg format and it takes a while to run the algorithm after uploading the picture.
